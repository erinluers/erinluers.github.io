Making your first GitHub webpage

System Setup

Setting up your system properly is very important. Make sure to use the references linked below, which provide detailed instructions.

  1. Download and set up VScode and make a free account.
  2. Generate SSH-Key using the command line. This video recommended by MITxPro is very helpful in walking through the steps.
  3. Make a free GitHub account.
  4. Add SSH-Key (public) to GitHub account settings. Watch video linked above for step-by-step instructions.
  5. Make a new GitHub Repository. In GitHub, select top right account icon and navigate to "your repositories". Select the green "New" button. Note: repository name cannot be random for GitHub to host the website, it must be named "username.github.io".

Connect VScode and GitHub Repository

  1. Clone GitHub Repository in VScode
    • Go to GitHub repository and click the green "code" button to copy the SSH code.
    • In VScode, open the terminal and navigate to directory you wish to copy repository to for your local copy.
    • In command line, enter "git clone SSH-code". Note: paste the SSH-code that you copied from the github repository.
  2. Make webpage content in html file
    • Baseline Requirements. In VScode, make sure you are still in the cloned directory on local drive (pwd) and make a new file "index.html" with website content
    • Make it Fancy. It is best practice to have separate files for CSS and JS. If you do separately and not in-line edits, don't forget to link the files to the index.html file!
      • add a CSS file for design.
      • add a JS file for interactivity
  3. Sync html file and GitHub Repository. In VScode terminal:
    • Add new html to GitHub.
      • git status to check what is in the folder and the current file connections with GitHub
      • git add . (for all listed files) or git add filename (for specific files) to add the file to GitHub
    • Commit change to GitHub
      • git status to check the added files are set in GitHub. They should indicate that they are a "new file" and be green.
      • git commit -m "description of changes in this version made" to commit the version control changes to GitHub
    • Push change to GitHub
      • git push origin main to actually update the changes in the remote GitHub repository

Open GitHub webpage!

  1. Open your preferred web browser and navigate to "username.github.io". Note: after pushing changes from VScode to GitHub, this can sometimes take a minute to refresh
