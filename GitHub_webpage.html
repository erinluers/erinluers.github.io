Making your first GitHub webpage

System Setup

  1. Download and set up VScode and make a free account
  2. Generate SSH-Key using the command line
  3. Make a free GitHub account
  4. Add SSH-Key (public) to GitHub account settings
  5. Make a new GitHub Repository

Connect VScode and GitHub Repository

  1. Clone GitHub Repository in VScode
  2. Make webpage content in html file
    1. Baseline Requirements
    2. Make it Fancy
  3. Sync html file and GitHub Repository
    1. Add new html to GitHub
    2. Commit change to GitHub
    3. Push change to GitHub
