Projects Overview

  • Creating a GitHub Webpage

    • Learn how to create a github webpage starting from the very beginning. Step by step.

  • Pacman Exercise

    • Learn how to use JS to pull an image of a pacman and have it move across the screen and bounce back and forth.

  • Eye Exercise

    • Learn how to incorporate user interactivity in your code. See how the eyes move to follow where the user puts their cursor.

  • Real Time Bus Tracker

    • Learn how to incorporate a token from an external source to gather data and then do what you want with it. In this instance, we pull bus information and track the real time location of the bus between MIT and Harvard.
